== Description ==

= German (de-DE) =

Die Datensammelleidenschaft von sozialen Netzwerken wie Facebook, Twitter, Google+, LinkedIn und anderer, ist bekannt. Wichtige Werkzeuge f&uuml;r die Datenerhebung sind die jeweiligen Social Sharing Buttons, die auf einfache Weise in allen Arten von Webseiteninstalliert werden k&ouml;nnen. Bereits beim Aufruf der Webseite werden dabei Userdaten gesammelt und an die Netzwerke &uuml;bertragen. 
Dieses Verhalten erh&ouml;ht die Ladezeit der Seite bei vielen Buttons deutlich. Dazu kommt, dieses Verhalten widerspricht geltendem deutschen Recht, da der Nutzer hierbei keine M&ouml;glich der Ablehnung oder Zustimmung hat.

Das Wordpress Plugin BST share iT sendet beim Teilen einer Seite keine Nutzerdaten ohne Einwilligung an die sozialen Netzwerke. Die Z&auml;hlerdaten werden in der Datenbank gespeichert und von dort abgerufen. Synchronisiert werden die Z&auml;hler &uuml;ber Scripte auf dem Webserver. Die Netzwerke sehen nur die IP vom Provider.
Derzeit verf&uuml;gt BST share iT &uuml;ber Buttons f&uuml;r Facebook, Twitter, Google+, LinkedIn, StumbleIn, Pinterest und XING. Auch ein Mail Button ist integriert. Die Buttons k&ouml;nnen in Seiten und Beitr&auml;gen sowie mit einem Widget in Sitebars angezeigt werden.

Holen Sie sich ihr BST share iT jetzt.

= English (en-EN) =

The data collecting mania of social networks like Facebook, Twitter, Google+, LinkedIn and others is well known. Important tools for those data collection are the respective social sharing buttons that can be installed in simple manner in all kind of websites. Already when calling the website the Buttons collected and transmitted user data to the networks.

This behavior increase the load time of the page significantly. However, it is also contrary to German law.

The BST share iT WordPress Plugin does not send user data to the social networks while sharing a page. The counter data are temporarily stored in the database and retrieved from there. BST share iT get this data via scripts on the Web server. The networks see only the IP of the provider.
Currently has BST share iT Buttons for Facebook, Twitter, Google+, LinkedIn, Stumble, Pinterest and the German XING. Also a mail button for sharing is available. Buttons can be displayed in pages and posts and with a widget in sidebars.

Get BST share iT now.


== Installation ==

1. Upload the BST share iT folder to the `/wp-content/plugins/` directory
2. Activate the BST share iT plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the 'BST share iT Buttons' menu that appears in your admin menu
